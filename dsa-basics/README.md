# DSA Basics

This folder contains beginner-level DSA programs:

- max_arr.java – find maximum element (O(n))
- CountVowels.java – count vowels in a string (O(n))
- CheckPalindrome.java – check palindrome (O(n))
- FindSecondLargest.java – find second largest element
- CountWords.java – count number of words

These programs help build a strong foundation for coding interviews.
